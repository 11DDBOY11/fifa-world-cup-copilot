# BC Place — Gate Map / Plan des Portes

## Overview / Aperçu
BC Place features four main entrances for general public: Gate A, Gate B, Gate C, and Gate D. All signage inside the stadium is bilingual in English and French.

## Gate Locations / Emplacements
- **Gate A**: North-West entrance. Nearest to Terry Fox Plaza. 
- **Gate B**: North-East entrance. Closest to the Vancouver Public Library.
- **Gate C**: South-East entrance. Closest to the accessible drop-off zone.
- **Gate D**: South-West entrance. Closest to the water shuttle terminal.

Gates open 2 hours before the event starts.
