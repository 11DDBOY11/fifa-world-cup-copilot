# Estadio Azteca — Mapa de Accesos

## Resumen
El Estadio Azteca cuenta con cuatro accesos principales numerados: Puerta 1, Puerta 2, Puerta 3 y Puerta 4.

## Ubicación de Puertas
- **Puerta 1**: Lado Norte. Conveniente para quienes llegan en Tren Ligero.
- **Puerta 2**: Lado Este. Cerca de las zonas de comida y mercancía oficial.
- **Puerta 3**: Lado Sur. Acceso recomendado para personas con discapacidad debido a la cercanía con el estacionamiento accesible (a 50 metros).
- **Puerta 4**: Lado Oeste. Cerca de los palcos VIP y del acceso a prensa.

Las puertas abren 3 horas antes del partido.
