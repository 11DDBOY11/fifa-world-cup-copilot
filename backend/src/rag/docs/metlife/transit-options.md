# MetLife Stadium — Transit Options

## Meadowlands Rail Line
- Recommended transit option. Trains run directly to MetLife Stadium Station from Secaucus Junction.
- Secaucus Junction connects to New York Penn Station (approx. 7 miles away).
- Trains run every 10-15 minutes on match days.

## Shuttle & Bus
- Bus 351 Express runs between the Port Authority Bus Terminal in NYC and MetLife Stadium.
- Shuttle buses operate from Lot B to remote parking lots.
