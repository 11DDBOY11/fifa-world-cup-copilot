# MetLife Stadium — Gate Map

## Overview
MetLife Stadium has four main public gates: Gate A, Gate B, Gate C, and Gate D. 

## Gate Locations
- **Gate A**: Located on the North side. Closest to Parking Lot A and the Meadowlands Rail Station.
- **Gate B**: Located on the East side. Closest to the Fan Zone and merchandise tents.
- **Gate C**: Located on the South side. Nearest to Parking Lot C (designated ADA parking).
- **Gate D**: Located on the West side. Closest to VIP check-in and the media entrance.

All gates open 2 hours before kickoff.
